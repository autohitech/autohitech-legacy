## 테이블 상세 정보
기술적인 컬럼 타입, Null 허용 여부, 기본값 등 상세 정보는 [DATABASE_SCHEMA.md](./DATABASE_SCHEMA.md) 문서를 참고해 주세요. 이 한국어 문서에서는 주요 테이블의 역할에 집중합니다.

### 그누보드 4 표준 테이블
- **`g4_config`**: 전역 설정 관리
- **`g4_member`**: 회원 데이터 (비밀번호는 해시화되어 저장됨)
- **`g4_board`**: 게시판 설정 관리
- **`g4_write_*`**: 게시글 실제 데이터 (게시판 이름별로 테이블이 분리됨)

### 커스텀 및 레거시 테이블
- **`diary`**: 업무 일지 로그 (Mid 컬럼으로 회원과 연결됨)
- **`customers`**: CRM 고객 연락처 정보
- **`adm_user`**: 별도의 관리자 인증 정보
- **`bbs`**: 과거 사용된 게시판 잔재 데이터
