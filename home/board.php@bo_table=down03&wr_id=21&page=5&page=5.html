    안녕하십니까. 기술지원팀입니다.

    각 태그별로 설정한 항목의 태그값들을 CSV 파일 형태로 저장하고 불러오는 방법에 대한 예제입니다.

    조명제어등에서 사용하는 각 그룹별로 조명의 밝기 설정한 값을 일괄적으로 적용하는 경우와 같이 적용이 가능하며 그 해당 그룹이나 시간, 날짜등을 파일 이름으로 저장하고 불러오기하여 적용이 가능합니다.

    작업선택 프로그램의 글자메뉴 "파일 | 복원" 메뉴를 이용하시어 첨부파일을 지정하시면 예제 작업이 작업선택 프로그램에 추가됩니다.

    예제 작업 버전 : AutoBase 9.3.2
    (예제 작업 이후의 버전을 사용하시기 바랍니다.)
